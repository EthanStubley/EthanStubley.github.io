Wordle & LED Design Project

- Creating Wordle and a LED Arduino game in C

View full project report here


For the Computer Systems and Architecture Module in my second year, I had to complete a two part design project, for the first part I created a Wordle game which used 20 sample words instead every 5 letter word in the English Language, but aside from that, it functioned just like the original Wordle game from the New York Times. For the second part of this project I made use of the a modified Arduino Development Board which was supplied by the University to creat an LED reaction game, where you had to press the button on the board in a set time frame after an LED lit up.

The source code and further details on this project can be found in the attched Documentation on this page
